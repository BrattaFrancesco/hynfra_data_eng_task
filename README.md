# Hynfra Data Engineering Task

## Overview
This repository contains a data engineering task focused on processing and analyzing energy price and weather data. Pre-run notebooks are available in the artifacts folder in HTML and PDF formats, allowing you to analyze the code and results without installing a Python environment or running the notebook.

## Installation
To run the notebook, create a Python virtual environment and install the required dependencies listed below.

## Usage
To analyze the results, view the pre-run notebooks in the artifacts folder. To run the code yourself, set up the environment as described and execute the task.ipynb notebook.

## Data
The input data files (prices1.csv, prices2.csv, weather.csv, etc.) are located in the data/ directory. The final processed datasets (national_features.csv, city_features.csv, city_hybrid_features.csv) are available in data/output for analysis.

## Requirements
- Python 3.x
- pandas
- numpy
- jupyter
